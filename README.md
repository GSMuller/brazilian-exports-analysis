# Brazilian Exports Analysis Dashboard

Interactive analytics dashboard for Brazilian export data (Comex Stat - MDIC).

## Overview
Analysis of Brazilian export performance by NCM, country destination, transport mode, and region.

## Data Source
- **Source:** Comex Stat (MDIC)
- **Period:** December 2024
- **Records:** ~50k transactions (sample from full dataset)

## Features
- Export volume by product (NCM)
- Top destination countries
- Transport mode analysis
- Regional distribution (UF)
- Time series trends

## Tech Stack
- Python 3.11
- PostgreSQL
- Streamlit
- Plotly
- Pandas

## Setup
```bash
pip install -r requirements.txt
streamlit run src/dashboard.py
```
